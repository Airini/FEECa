Points

The Point provides the Point data type that is used to represent positions in n-dimensional Euclidean space.

The Point type

The Point data type is build on top of the Vector data type using a Vector to represent the coordinates of the given point.

-- | Points in n-dimensional Euclidean space. A point describes a fixed position
-- | in space and can not be computed with.
newtype Point a = Point (Vector a) deriving (Eq, Show)

instance Dimensioned (Point a) where
    dim (Point v) = dim v

instance Real a => Pretty (Point a) where
    pPrint (Point v) = text "Point in "
                       <> rn (dim v)
                       <> text ":\n"
                       <> printVector 2 (map (fromRational . toRational) (components v))

-- | Constructor for the Point data type.
point :: [a] -> Point a
point = Point . vector

Convenience Functions

The Point data type provides convenience functions for the creation of special points in \({\mathbb{R}^{n}}\). The zero function returns the Point instance representing the origin in \({\mathbb{R}^{n}}\). unit returns the point in \({\mathbb{R}^{n}}\) that lies at a distance of 1 from the origin on the given axis.


-- | Return the origin in R^n.
-- zero :: Num a => Int -> Point a
-- zero n = Point $ vector (replicate n 0)

-- -- | Point with the ith component 1.0 and all other components 0.0.
-- unit :: EuclideanSpace v => Int -> Int -> Point (Scalar v)
-- unit n i = Point (unitVector n i)

Conversion to and from Vectors

The point data type doesnâ€™t provide any arithmetic functions. In order to perform computations with points it is necessary to convert them to vectors. The vector corresponding to a given point is just the vector pointing from the origin to the given point.


-- | Point corresponding to given position vector.
toPoint :: Vector a -> Point a
toPoint = Point

-- | Position vector of given point.
fromPoint :: Point a -> Vector a
fromPoint (Point v) = v

-- Instances
instance Module (Point Double) where
  type Scalar (Point Double) = Double
  addV x = toPoint . uncurry addV . pairM fromPoint fromPoint . (x,)
  sclV _ = undefined

instance VectorSpace (Point Double)

instance EuclideanSpace (Point Double) where
  dot x    = uncurry dot . pairM fromPoint fromPoint . (x,)
  fromList = point
  toList   = toList . fromPoint
